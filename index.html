Satchless

Less is mo'

E-commerce.
For Python.

Satchless brings e-commerce and Python together. It provides the low-level classes and patterns so you can focus on your business logic and user experience.

It's free as in beer, well-tested and framework-agnostic.

Python, meet Satchless

No frameworks attached: No matter if you're building your store from scratch or using a popular framework like Django or Flask, you can use Satchless and let us worry about the low-level logic.

It's all about interfaces: Easily bind the desired behaviour to a Django model, a SQLAlchemy record or a pure Python object and have Satchless do the Right Thing™.

Proven to work: the library comes with an extensive test suite that covers every line of Python code. You can check our scores on Travis CI and Coveralls.

Future-proof: Satchless comes with support for Python versions 2.6, 2.7, 3.2 and 3.3.
